# Скрипты Duster

## install.ps1 - Windows

Единый установщик для Windows. Запуск от администратора.

```powershell
.\install.ps1
```

Режимы:
- [0] Docker Compose
- [1] Сервер
- [2] Админ-панель
- [3] ПК в зале

## install.sh - Linux

Установщик для Linux (Ubuntu/Debian/CentOS).

```bash
sudo ./install.sh
```

## setup-nginx-certbot.sh

SSL сертификаты для Linux сервера.
